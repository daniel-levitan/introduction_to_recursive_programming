Definition: s_n = s_n-1 + 3

(a) s_0 = 0

s_1 = 0 + 3 = 3
s_2 = 3 + 3 = 6
s_3 = 6 + 3 = 9
s_4 = 9 + 3 = 12

(b) s_0 = 4

s_1 = 4 + 3 = 7
s_2 = 7 + 3 = 10
s_3 = 10 + 3 = 13
And so on

A non recursive definition would be:

(a)
s_n = 0          if n = 0
s_n = 0 + n * 3  if n > 0

(b)
s_n = 4          if n = 0
s_n = 4 + n * 3  if n > 0
